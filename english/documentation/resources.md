# Technical Resources

## About Mosaic

Mosaic is a decentralized borrowing protocol built on Reef Chain. Users can deposit REEF as collateral to borrow MEUR, a Euro-pegged stablecoin.

### Key Features

- **0% Interest Rate**: Borrow MEUR with no ongoing interest charges
- **110% Minimum Collateral Ratio**: Capital-efficient borrowing
- **Governance-Free**: Fully algorithmic and immutable
- **Euro-Pegged Stablecoin**: MEUR provides Euro-denominated stability
