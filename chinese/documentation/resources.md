# Technical Resources

## 关于Mosaic

Mosaic是一种基于Reef Chain的去中心化借贷协议。用户可以存入REEF作为抵押品来借入MEUR，一种与欧元挂钩的稳定币。

### 主要特点

- **0%利率**：借入MEUR无需支付持续利息
- **110%最低抵押率**：资本高效的借贷
- **无治理**：完全算法化和不可变
- **欧元挂钩稳定币**：MEUR提供欧元计价的稳定性
